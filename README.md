# マインスイーパー 

vue-cli環境で、マインスイーパー（的なもの）を作成しました。

## 使い方

Vue CLIをインストールします

```
npm install --global @vue/cli
vue create my-project-name
```

（参考）
https://jp.vuejs.org/v2/guide/typescript.html

その後、「my-project-name」の位置に、このgitの内容を展開してください。
